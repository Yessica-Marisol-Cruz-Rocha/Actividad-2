GIT Y GITHUB

¿Qué es GIT?

Git es un sistema de control de versiones distribuido, o sea registra los cambios en los archivos o códigos todo el tiempo. Da seguimiento a las modificaciones y mantiene el historial de cambios en el repositorio local de cada desarrollador, por lo cual se tiene acceso a diferentes versiones.

¿Qué es GITHUB?

Github es el repositorio de estos archivos, almacena las versiones de código y al ser online permite el trabajo cooperativo.
